# Smart Cart – Customer Segmentation System

A data-driven unsupervised learning project that segments customers based on behavior to enable targeted marketing and improved business decision-making.

### 🛠️ Technical Highlights
- **Clustering Algorithms:** Applied **K-Means** and **Hierarchical Clustering** to identify distinct customer personas.
- **Optimization:** Used the **Elbow Method** and **Silhouette Analysis** to determine the optimal number of clusters.
- **Dimensionality Reduction:** Implemented **Principal Component Analysis (PCA)** to simplify complex datasets while retaining key variance.
- **Insights:** Generated actionable behavioral insights to drive data-driven marketing strategies.

### 📊 Tech Stack
- Python (Jupyter Lab)
- Scikit-learn
- Pandas & NumPy
- Matplotlib & Seaborn
